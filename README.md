# cnctd_roku

Roku device integration library for Rust.

## Installation

```toml
[dependencies]
cnctd_roku = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Features

- Device discovery (SSDP)
- Remote control commands
- App launching
